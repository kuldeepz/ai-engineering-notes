Each guide is a standalone HTML file. Open in any browser, no dependencies.

## How to use

Clone and open any `index.html` directly in a browser — no server needed.

```bash
git clone https://github.com/kuldeepz/ai-engineering-notes
open ai-engineering-notes/01-rag-knowledge-platform/index.html
```
